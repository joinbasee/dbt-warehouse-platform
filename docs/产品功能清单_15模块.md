# 智能数仓建设平台 — 产品功能清单

> PEP 企业数据仓库 | 15 个功能模块 | 2026-07-02

---

## 一、平台概述

智能数仓建设平台是一个 **AI 驱动的数据仓库建设系统**，由三层组成：

- **Electron 桌面应用**：统一操作面板（React + Vite + Recharts）
- **Python 脚本层**：SQL 生成 / 字段盘点 / ETL 编排 / 文档导出
- **dbt 核心引擎**：14 个标准化 Jinja 宏 + 数据质量验证

目标：将 SAP ECC 的 ABAP 报表逻辑迁移到 Hive + OceanBase 数仓，供 FineReport BI 消费。

### 模块 14：自动化部署

| 属性 | 内容 |
|------|------|
| **入口** | GitHub Actions + DolphinScheduler + Electron Builder |
| **源码** | `.github/workflows/dbt-ci.yml` / `.github/workflows/dbt-deploy.yml` / `scheduler/` / `platform/package.json` |

#### 第 0 层：环境标准化
- `profiles.yml.template` — 所有密码替换为 `{{ env_var() }}`
- `.env.example` — 环境变量模板
- `scripts/setup_env.py` — 一键生成 `profiles.yml`（支持 `--check` / `--ci` 模式）

#### 第 1 层：CI 门禁（每次 push / PR）
- `dbt parse`（语法校验，秒级完成）
- `dbt test --target dev`（DuckDB 本地测试）
- 配置文件：`.github/workflows/dbt-ci.yml`

#### 第 2 层：预发布验证（合并到 master / 手动触发）
- `dbt run --target hive`（Hive 生产执行）
- `dbt test --target hive`（生产数据质量）
- `scripts/validate_cross_db.py`（跨库对比）
- Webhook 通知（飞书/企业微信）
- 配置文件：`.github/workflows/dbt-deploy.yml`

#### 第 3 层：定时调度（DolphinScheduler / cron）
- 5 个业务场景的完整 DAG 定义
- 统一任务脚本模板 `dbt_task_template.sh`
- 告警策略（任务失败 / 超时 / 数据差异）

#### 第 4 层：Electron 打包分发
- `npm run build:win` → `.exe` NSIS 安装包
- 桌面快捷方式 / 自定义安装路径
- 配置：`platform/package.json`

### 模块 15：标准化宏引擎

| 属性 | 内容 |
|------|------|
| **入口** | Jinja 宏 |
| **源码** | `macros/*.sql`（14 个文件） |
| **功能** | 所有 DWD/DWS 模型必须使用的标准宏，禁止手写原始 SQL |

| 宏 | 用途 |
|----|------|
| `sap_date(col)` | yyyyMMdd 字符串 → DATE 类型 |
| `sap_string(col, default)` | TRIM + COALESCE 默认值 |
| `sap_flag(col)` | SAP 'X'/' ' → '1'/'0' |
| `sap_decimal(col, precision)` | NULL→0 + CAST DECIMAL |
| `audit_columns()` | 统一 7 列审计字段 |
| `incremental_filter(col, days)` | 增量加载 WHERE 子句 |
| `group_concat(expr, sep)` | 跨方言 GROUP_CONCAT（DuckDB / Spark / OceanBase） |
| `generate_schema_name()` | 自定义 schema 命名 |
| `zsdr005_params()` | ZSDR005 报表参数化 |
| `compare_ob()` | OceanBase 跨库行数对比 |
| `describe_table()` | 跨库表结构查询 |
| `alter_add_columns()` | 动态 ALTER TABLE 加列 |
| `create_billing_detail()` | 开票明细宽表创建 |
| `get_columns()` | 获取表字段列表 |
