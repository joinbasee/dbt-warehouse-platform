## 统一规范

- **分辨率**: 1440×920（Electron 窗口默认尺寸）

## 操作步骤

1. 双击 `platform/start.bat` 启动应用
2. 等待 Electron 窗口出现，确认左侧栏底部显示"系统运行中"
3. 按 `F11`（全屏）或保持默认 1440×920 窗口
4. 切换到目标页面，等待数据加载完成
5. 按 `Win + Shift + S` → 选择"窗口截图"模式
6. 保存到 `screenshots/` 目录，按上表命名
